# TextTanuki

A simple API that analyzes a piece of text and finds the similarity percentage between two texts. 

The analysis tool provides information such as the number of characters, sentences, word count, most frequent word and the longest word in a single text.

First install the necessary packages using

```npm install```
  
Then start the server using 

```node app.js```

To run unit tests on Mocha and Chai use

```npm run tests```
